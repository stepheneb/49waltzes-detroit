Keyboard interface in Map page

The keyboard interface works in the map page when it is not in full-screen mode.

alt/option-T: Toggle testing mode on or off

Switching video resolution

  • alt/option-1: 480x270
  • alt/option-2: 960x540
  • alt/option-3: 1920x1080

Test pages:

  • video test: 480x270 500kbps
  • video test: 960x540 2Mbps
  • video test: 1920x1080 10Mbps

The uncompressed video file used for these transcoding tests was truncated to 1/3 the length of the original (640 instead of the original 1920 frames).
